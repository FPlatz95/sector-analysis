Hvad skal værktøjet kunne?
Det skal kunne bruges som en analyse af specifikke brancher samt bruges til at undersøge en eller flere virksomheders bidrag til branchen. 
Analyse af en branche:
- Total antal medarbejdere

Analyse af en eller flere virksomheders bidrag til branchen:
- Liste over samtlige virksomheder i en branche
- Antal medarbejdere samt 

Trin for at kunne gøre dette:
- Prøv at hente data ned og lave nogle analyser
- Design pipelines der kan tage parametre og hente dataen ned.
